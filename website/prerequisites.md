# Workshop Prerequisites

- **Laptop:** Bring your own laptop. All tutorials run on CPU — no GPU required.
- **Python:** Experience with Python 3, virtual environments, and package management.
- **Jupyter:** Experience with Jupyter notebooks or Google Colab.
- **ML frameworks:** Some of the tutorials will use PyTorch. Hence, some familiarity will be good, although a quick rundown will be provided. Attendees are encouraged to be familiar with concepts presented in the PyTorch [intro tutorials](https://docs.pytorch.org/tutorials/beginner/basics/intro.html).
- **Afternoon hands-on:** The afternoon session includes building a neural controller for detector control. All materials and starter code will be provided.

Some dedicated time for setup is expected on Sept 24; details will be provided.
